# CodeCraft Technologies - Interactive Landing Page 🚀

A modern, fully responsive tech startup landing page featuring an interactive navigation menu. This project was developed as part of **Task 1** during the **Web Development Internship** at **Prodigy InfoTech**.

---

## ✨ Features Implemented

* 🧭 **Fixed Navigation Bar** — Stays anchored at the top of the viewport for easy, continuous access while scrolling.
* 🎨 **Dynamic Scroll Effect** — Detects window scrolling via JavaScript. When users scroll past `50px`, the navbar seamlessly transitions from a soft pastel Lilac (`#A55EEA`) to a beautiful Blossom Pink (`#FF8DA1`).
* ✨ **Smooth Transitions** — Features custom CSS interactions that dynamically scale text links slightly and smoothly change colors during hover states.
* 📑 **Smooth Anchor Scrolling** — Links directly to single-page sections (`#home`, `#services`, `#features`, `#about`, `#contact`) powered by fluid native CSS scroll behavior.
* 📱 **Mobile-First Responsiveness** — Implements media queries (`@media max-width: 768px`) to vertically stack layout elements and collapse navigation cleanly across tablets and smartphones.

---

## 🛠️ Built With

* **HTML5** — Semantic page structuring and layout.
* **CSS3** — Custom typography, flexbox formatting, transitions, and soft pastel styling.
* **Vanilla JavaScript (ES6)** — Scroll event tracking and dynamic class list modifications.

## 🚀 How to Run Locally

Follow these quick steps to run the landing page on your local computer:

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```
2. **Navigate to the project directory:**
   ```bash
   cd Navigation_Menu
   ```
3. **Open the project:**
   * Double-click the `index.html` file to launch it directly in your default browser.
   * *Alternative:* Open the folder in VS Code and launch it using the **Live Server** extension for real-time changes.
